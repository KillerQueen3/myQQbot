# 测试用QQ机器人No.1
基于 [mirai](https://github.com/mamoe/mirai) 的qq机器人。
## 主要功能
* 查看今天课表
* 发送一些网络文本
* 复读
* 发送随机本地图片(./resource/image/标签/ 下的图片，默认使用./resource/image/default/ 下的图片)
* 发送随机pixiv图片，使用[api](https://api.lolicon.app/setu/)
* 查看pixiv部分榜单
* 以pixiv id找图
* 以pixiv用户id发送用户随机作品
* ...
